## What is Xoxemons?

Xoxemons is a full-stack web application that combines creature-collecting mechanics with PVP risk-reward combat. Every player starts with a small collection and builds it over time by claiming daily rewards, winning battles, and keeping their creatures healthy.

## Core features

- **Growth stages** — Xoxemons evolve from Small to Medium (3 sweets) and Medium to Large (5 sweets).
- **Disease system** — Feeding carries RNG risk. A sick Xoxemon needs +2 extra food to evolve and must be cured with a vaccine.
- **Battle engine** — Combat is decided by a dice roll, creature size bonus, and elemental type matchup (Water beats Earth, Earth beats Air, Air beats Water).
- **Daily reward** — Log in daily to claim one random Xoxemon and 10 sweets.
- **Admin dashboard** — Administrators can give items or Xoxemons to any player and tune global disease probabilities.

## Getting started

<Steps>
  <Step title="Clone and configure">
    Clone the repository and copy the backend environment file from `.env.example`.
  </Step>
  <Step title="Start the containers">
    Run `docker-compose up -d --build` from the project root. This builds and starts the Angular frontend, Laravel backend, and MySQL database.
  </Step>
  <Step title="Seed the database">
    Enter the backend container and run `php artisan migrate:fresh --seed` to create tables and load the default Xoxemons, items, and admin user.
  </Step>
  <Step title="Open the game">
    Navigate to `http://localhost:4200` to register your account and claim your first daily reward.
  </Step>
</Steps>

See the [full quickstart guide](/quickstart) for detailed commands and troubleshooting tips.
